# Граф связей

## Цвета узлов

- <span style="color:#2E7D32">**Светло-зелёный**</span> — источник загрузки: только исходящие связи, отсюда контент берётся.
- <span style="color:#1565C0">**Светло-синий**</span> — цель публикации: только входящие связи, сюда контент заливается.
- <span style="color:#B26A00">**Кремовый**</span> — транзит: и входящие, и исходящие связи (например, HDD-кеш посередине).
- <span style="color:#757575">**Серый**</span> — отключённый источник.

## Режимы

- **Обычный** — левый клик по узлу подсвечивает его и соседей, повторный клик или клик по пустому месту — сброс.
- **Создание связи** — включается кнопкой «Создать связь» в тулбаре или пунктом «Создать связь отсюда» в меню правой кнопки по узлу. Два клика по узлам создают связь. После создания режим сохраняется с тем же источником — можно быстро добавить несколько связей подряд.

## Контекстные меню

- **Правый клик по узлу** — «Создать связь отсюда».
- **Правый клик по ребру** — «Инвертировать», «Удалить».

## Горячие клавиши и кнопки

- **Esc** — выйти из режима создания связи.
- **Кнопка «Перестроить»** — заново читает связи из базы и пересобирает раскладку графа (сбрасывает текущий зум).
