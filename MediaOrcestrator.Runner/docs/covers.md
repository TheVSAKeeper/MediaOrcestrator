# Генератор обложек

Автоматически делает обложки для группы видео по общему шаблону. Полезно, когда нужно поставить на 20 серий подряд одинаковые превью с разными номерами — не руками в фотошопе, а одним кликом.

Используется в окне **Обновление превью** (правый клик по нескольким медиа → «Обновить превью»). Там появляется режим **«Из шаблона»** рядом с «Из источника» и «Из файла».

## Как пользоваться

1. В таблице медиа выделите нужные видео (Ctrl+клик или Shift+клик)
2. Правый клик → «Обновить превью»
3. Включите режим **«Из шаблона»**
4. Нажмите **«Настроить…»** — откроется редактор шаблона
5. Выберите картинку-шаблон, расставьте слои, нажмите **OK**
6. Выберите целевые площадки в списке «Куда загрузить»
7. **«Применить»** — на каждое видео генерируется своя обложка и загружается на выбранные площадки

Сгенерированные обложки живут в `temp_path/{guid}/cover_{номер}.png` и удаляются после загрузки.

## Шаблон обложки

**Файл шаблона** — обычная картинка (PNG / JPG / WEBP), поверх которой рисуются слои с текстом. Разрешение любое — слои масштабируются от высоты картинки, обложка корректно отрисуется и на 1280×720, и на 1920×1080.

**Слои** — произвольное количество текстовых надписей. Каждый слой имеет свои координаты, шрифт, размер, цвет заливки, обводку. Рисуются по порядку сверху вниз по списку — верхние элементы в ListBox рисуются первыми, нижние — поверх.

### Редактирование слоя

Выделите слой в списке «Слои» — снизу активируется группа «Текущий слой»:

- **Текст** — строка с возможным плейсхолдером `{number}`. Примеры: `{number}`, `Серия {number}`, `EP.{number}`, `ФИНАЛ`
- **Шрифт** — любой установленный в системе
- **Размер** — в процентах от высоты картинки (25% — крупный, 10% — мелкий)
- **Обводка** — толщина в процентах от высоты; 0% = без обводки
- **Заливка** / **Цвет обводки** — кнопки открывают стандартный диалог цвета

### Добавить слой / Удалить

Кнопки справа от списка. Новый слой создаётся с дефолтными параметрами (белый текст, чёрная обводка, центр по X, сверху по Y). Удалить последний оставшийся слой нельзя.

### Позиционирование

**Клик левой кнопкой мыши по превью** задаёт позицию выбранного слоя. Координаты хранятся как доли от размера картинки (0.5 × 0.5 = центр), поэтому работают одинаково для любого разрешения шаблона.

## Нумерация

Определяет, какое число подставится вместо `{number}` в каждом слое для каждой обложки.

### Последовательно

Простой режим — каждой обложке присваивается номер `Начальный № + позиция в выборке`. Если выбрано 5 видео и начальный номер 3, то обложки получат номера 3, 4, 5, 6, 7. Порядок соответствует порядку медиа в гриде.

### Из названия (regex)

Полезно, когда у видео в базе уже есть номер в заголовке — например «Бэкпекинг по Казахстану #12». Генератор вытаскивает номер регуляркой из `Media.Title` и подставляет в шаблон.

**Поле Regex** — регулярное выражение, первая группа захвата используется как номер. Если групп нет — используется всё совпадение. Если совпадения нет или в заголовке нет цифры — фолбэк на `Начальный № + позиция`.

**Примеры регулярок:**

- `#?(\d+)` — универсальный, берёт первое число, опционально после `#`
- `[Сс]ерия\s+(\d+)` — только после слова «Серия»
- `EP\.?\s*(\d+)` — только после «EP» / «EP.»
- `\((\d+)\)` — число в скобках, например «Финал (13)»

**Если регулярка невалидная или зависает** (таймаут 100 мс) — в логи пишется warning, используется фолбэк. Пакет не падает.

## Профили

Редактор умеет сохранять настройки целиком (файл шаблона, режим нумерации, regex, все слои, цвета, позиции) в именованный профиль. Хранится в `{папка с settings.txt}/templates/covers/{имя}.json`.

### Сохранить как

Кнопка **«Сохранить как…»** в группе «Профиль» — вводите имя, сохраняется. Если профиль с таким именем уже есть — спросит подтверждение перезаписи.

### Загрузить

Выберите профиль в комбобоксе, нажмите **«Загрузить»** — форма переоформляется целиком под загруженные настройки. Несохранённые текущие изменения теряются.

### Удалить

Кнопка **«Удалить»** — спросит подтверждение, удалит JSON из папки профилей.

### Быстрое переключение

В окне «Обновление превью» рядом с кнопкой «Настроить…» есть свой комбобокс профилей — можно переключиться между готовыми профилями одним движением, не заходя в редактор.

### Автосохранение `last`

Каждый раз при нажатии OK в редакторе текущее состояние сохраняется в скрытый профиль `last`. При следующем открытии формы он подгружается автоматически — удобно, чтобы продолжить с того места, где остановились. В списке профилей `last` не показывается.

## Примеры

### Нумерованные серии одного сезона

- **Шаблон:** фон с лого шоу
- **Слой 1:** `{number}` в правом нижнем углу, крупно, белый с чёрной обводкой
- **Нумерация:** Последовательно, начальный № = 1

### Финальная серия отдельно

- **Шаблон:** тот же фон
- **Слой 1:** `ФИНАЛ` по центру, крупно
- Сохранить как профиль «Финал», переключиться перед применением

### Парсинг номеров из готовых названий

Медиа в базе: `Марафон #01`, `Марафон #02`, … `Марафон #15`:

- **Слой 1:** `Серия {number}`
- **Нумерация:** Из названия, regex `#(\d+)`
- Каждая обложка получит свой номер, даже если в выборке пропущены серии

### Две надписи на одной обложке

- **Слой 1:** `Серия {number}` в верхнем левом углу
- **Слой 2:** `МЕДИА ОРКЕСТРАТОР` внизу, жёлтый, крупнее

## Нюансы и грабли

- **Порядок в выборке важен.** В режиме «Последовательно» номер зависит от позиции видео в таблице. Если отсортировать грид иначе — номера поедут. Проверяйте порядок перед нажатием «Применить».
- **`{number}` — единственный плейсхолдер.** Подстановка `{title}` / `{date}` пока не поддерживается.
- **Прозрачность недоступна.** Стандартный ColorDialog не предлагает альфа-канал, все цвета непрозрачные.
- **Шрифт по умолчанию — Impact**, если он установлен. Иначе Arial Black, иначе — первый в списке.
- **Шаблоны зависят от абсолютного пути к картинке.** Если перенесёте файл шаблона — профиль перестанет работать, пока не укажете новый путь в редакторе.
