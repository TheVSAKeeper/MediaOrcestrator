# Настройка Telegram API для MediaOrcestrator

Для работы с каналами Telegram (чтение, загрузка, выгрузка и удаление видео) необходимо получить API-credentials
на my.telegram.org.

## 1. Получение API ID и API Hash

1. Перейдите на [my.telegram.org](https://my.telegram.org) и войдите по номеру телефона.
2. Выберите раздел **API development tools**.
3. Если приложение ещё не создано, заполните форму:
    - **App title** — только латинские буквы, формат UpperCamelCase (например, `MediaOrcestrator`).
    - **Short name** — аналогично title, 5–32 символа, только латиница без подчёркиваний, тире и пробелов
      (например, `MediaOrcestrator`). Можно совпадать с title.
    - **URL** — любой действующий URL (например, `https://github.com/TheVSAKeeper/MediaOrcestrator`).
    - **Platform** — выберите **Desktop**.
    - **Description** — развёрнутое описание приложения, **минимум ~40 символов**
      (например, `Desktop application for managing and synchronizing media content across platforms`).
4. Нажмите **Create application**.
5. Скопируйте **App api_id** (числовой) и **App api_hash** (строка).

> **Важно**: API-credentials привязаны к аккаунту Telegram. Одному аккаунту соответствует одно приложение.
> Удалить или пересоздать приложение нельзя — только отредактировать.

### Устранение ошибки «ERROR»

Форма на my.telegram.org возвращает только «ERROR» без пояснений. Возможные причины:

- **Пустые или короткие поля** — URL и Description обязательны. Description должен быть развёрнутым (~40+ символов).
- **Спецсимволы в названиях** — в App title и Short name допускаются только латинские буквы (a–z, A–Z), без
  подчёркиваний, тире, цифр и пробелов.
- **VPN** — если VPN включён, а номер телефона зарегистрирован в другой стране, создание приложения может блокироваться.
  Отключите VPN или используйте VPN страны регистрации номера.
- **Молодой аккаунт** — аккаунт должен существовать минимум 2–3 дня и иметь минимальную активность
  (подписки на каналы, контакты).
- **Кэш браузера** — попробуйте режим инкогнито или другой браузер. Отключите расширения (AdBlock и т.п.).

## 2. Настройка в MediaOrcestrator

В настройках Telegram-источника укажите:

| Настройка            | Значение                                                                           |
|----------------------|------------------------------------------------------------------------------------|
| `api_id`             | App api_id из шага 5                                                               |
| `api_hash`           | App api_hash из шага 5                                                             |
| `phone_number`       | Номер телефона аккаунта Telegram (формат: `+7XXXXXXXXXX`)                          |
| `channel`            | Username канала (`@channel`) или числовой ID                                       |
| `speed_limit`        | *(необязательно)* Ограничение скорости скачивания, Мбит/с. Пусто — без ограничений |
| `upload_speed_limit` | *(необязательно)* Ограничение скорости выгрузки, Мбит/с. Пусто — без ограничений   |

Файл сессии Telegram хранится в общей папке состояний (`state_path` в настройках приложения) в поддиректории источника — например, `state/{source_id}/telegram.session`. Ранее настройка называлась `session_path` — при первом запуске существующие сессии автоматически мигрируются.

## 3. Авторизация

При первом подключении MediaOrcestrator запустит интерактивный процесс авторизации:

1. На указанный номер телефона придёт **код подтверждения** (в Telegram или по SMS).
2. Введите код в диалоговом окне приложения.
3. Если на аккаунте включена **двухфакторная аутентификация (2FA)**, будет запрошен облачный пароль.
4. После успешной авторизации сессия сохранится в файл `telegram.session` внутри папки состояний источника.

При следующих запусках повторная авторизация не требуется — сессия загружается из файла автоматически.

> **Важно**: Не передавайте файл сессии третьим лицам — он даёт полный доступ к аккаунту Telegram.

## 4. Требования к каналу

- Аккаунт должен быть **подписан на канал** (или быть его владельцем/администратором).
- Для загрузки и удаления видео аккаунт должен иметь **права администратора** канала.
- Канал можно указать как `@username` (публичный) или как числовой ID (приватный).
- Ссылки вида `t.me/...` генерируются только для каналов с username.

## 5. Зависимости

Для извлечения метаданных видео при выгрузке требуется **FFmpeg** (а именно `ffprobe`).
MediaOrcestrator автоматически скачает FFmpeg при первом использовании, если он не установлен.

Если `ffprobe` недоступен, видео загрузится без метаданных (длительность, разрешение, кодек).

## Лимиты

Telegram API накладывает ограничения на частоту запросов. При интенсивном использовании возможны временные блокировки
(FloodWait). В этом случае клиент автоматически ждёт указанное сервером время перед повторной попыткой.

| Ограничение                    | Значение                            |
|--------------------------------|-------------------------------------|
| Максимальный размер файла      | 2 ГБ (загрузка и скачивание)        |
| Скорость скачивания             | До ~20 Мбит/с (зависит от DC)      |
| Частота запросов                | Динамический лимит от Telegram      |
